Development setup
-----------------

Requires Open SDK 8.

- Type ``./setup-eclipse.sh`` from the root of the repository
- From eclipse:
  - ``Import`` in ``File`` menu
  - ``Import existing projects into workspace``
  - Select the root of this repository
